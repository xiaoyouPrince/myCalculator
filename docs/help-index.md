# myCalculator 帮助文档目录

这个项目把产品说明、工程复盘和练手文章都放进了 App 的 Help 菜单。它不是商业软件常见的信息架构，但适合这个业余工具项目：写代码、修问题、总结经验都沉淀在同一个地方。

## 项目说明

- `项目说明`：完整 README，覆盖项目结构、功能范围、运行方式、后续迭代方向和早期更新记录。
- `工时计算规则`：从 README 中提取的工时与加班计算规则，适合使用 App 时快速查看。

## 配套工具

- `浏览器扩展自动填报`：说明 `browser-extension` 的用途、安装方式、极简 JSON 数据格式、填写模式和网页匹配策略。

## 工程复盘

- `问题复盘：月视图跨月渲染`：记录月视图当前月/非当前月颜色错乱问题的复现、根因、修复原则和验证流程。

## 技术文章

- `SwiftUI 与 UIKit 心智模型`：从月视图 Bug 出发，对比 SwiftUI 和 UIKit 在复用、identity、状态生命周期上的心智模型差异。

## 维护原则

- 使用说明写在 README 或对应功能文档中。
- 线上/日常操作会用到的内容放进 Help 菜单。
- 修复复杂问题后写复盘，尤其是涉及 UI 状态、数据持久化、导出格式和浏览器扩展适配的问题。
- 文档中的验证命令应尽量可复制执行。
- 如果文档被 App 运行时读取，需要同步加入 Xcode Resources。
