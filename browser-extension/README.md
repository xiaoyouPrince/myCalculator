# Work Hours Autofill

Chrome/Edge/Arc Manifest V3 扩展，用于把 `myCalculator` 导出的极简 JSON 自动填入网页月度工时表单。

## 数据格式

使用应用里的“导出极简 JSON”，格式类似：

```json
[
  {
    "day": "2026-06-25",
    "startTime": "09:00",
    "endTime": "20:56"
  }
]
```

扩展会按 `day` 匹配当前页面里当月 1-31 日，并填写当日工时申报值。

## 安装

1. 打开 Chrome/Edge/Arc 的扩展管理页。
2. 开启开发者模式。
3. 选择“加载已解压的扩展程序”。
4. 选择本目录：`browser-extension`。

## 使用

1. 打开目标网页表单。
2. 点击扩展图标。
3. 导入应用导出的极简 JSON。
4. 选择填写模式：
   - `工作小时数`：按 `myCalculator` 的当前工时申报规则填写，工作时长小于 10 小时扣 1 小时，大于 10 小时扣 2 小时，等于 10 小时不扣
   - `上下班时间段`：例如 `09:00-20:56`
5. 点击“自动填写当前页”。

默认小时精度是 1 位小数。填写成功后会显示本次导入数据的月度汇总：记录天数、月工作时长、月工时申报、月加班时长和月有效加班。

## 匹配策略

当前版本是通用扫描：

- 优先扫描表格行和单元格里的日期文本。
- 识别 `1` 到 `31` 或 `2026-06-25` 这类日期。
- 在日期附近查找可写的 `input` 或 `textarea`。
- 有匹配数据才填写，没有数据会跳过。

如果目标网页的日期和输入框不是普通表格结构，需要针对该网页 DOM 再加专用适配。
